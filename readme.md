# PingDuino

Live web status panel that displays the number of online clients and the events
associated with a real button connected to and Arduino board.

## Technologies

+ Arduino
+ Python
+ Socket.io
+ Node.js


## Requirements

+ Python 2.7
+ SocketIO_client
+ Node.js
+ Socket.io


## Running the project

    cd server
    node server.js

Now go to the browser and open *web_client/index.html*.
