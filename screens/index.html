The Experiment

Node.js

  • Socket.IO: Connection (Web clients and Arduinos).
  • Event management.

Python

  • Arduino client that talks to Node.js (Using SocketIO_client).
  • Comunication with the Arduino through the serial port serial.
  • Event handling.

Arduino

  • Input (Buttons, switches and sensors).
  • Output (Speakers, lights, motors, etc).
  • Ouputs information to the serial port.
